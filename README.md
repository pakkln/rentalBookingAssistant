# Instagram Rental Booking Assistant
(เว็บไซต์สำหรับการจองเสื้อผ้าออนไลน์กับร้านในอินสตาแกรม)

## ข้อมูลโดยสังเขป
    ในปัจจุบันกระบวนการเช่าชุดจากร้านในแอปพลิเคชัน Instagram มักทำผ่านการตอบข้อความโดยตรง ร้านค้าจำเป็นต้องทำขั้นตอนซ้ำ ๆ เช่น อธิบายขั้นตอนการเช่า ตรวจสอบข้อมูลลูกค้า ส่งกฎการเช่า และ ตรวจสอบวันว่างของชุดด้วยตนเองทุกครั้ง 
    
    กระบวนการดังกล่าวทำให้เกิดภาระงานสูง ใช้เวลามาก และ เพิ่มความเสี่ยงต่อความผิดพลาด

    โปรเจ็คนี้จึงสพัฒนา frontend web เพื่อช่วยให้ขั้นตอนการเช่า เข้าใจง่าย, ใช้งานสะดวก, ลดภาระงานร้าน และ ลดข้อผืดพลาดที่อาจเกิดขึ้นจากการจัดการด้วยตัวเอง

-------

## ปัญหา
    ในการจองชุดผ่านข้อความ ทางร้านต้อง:
    - ตอบคำถามเรื่องวันว่างของชุดให้ลูกค้าทุกคน
    - บันทึกข้อมูลลูกค้า Instagram username ที่อยู่ ด้วยตนเอง
    - ส่งกฎและเงื่อนไขการเช่าซ้ำหลายครั้ง
    ขั้นตอนเหล่านี้ยังขาดประสิทธิภาพและทำให้เกิดข้อผิดพลาดได้ง่าย

## วิธีแก้ไข
    Instagram Rental Booking Assistant ทำให้ลูกค้าสามารถ
    - เลือกชุดและวันที่ต้องการจะเช่าโดยไม่ต้องรอร้านตอบข้อความ
    - อ่านและยอมรับกฎการเช่าครบทุกข้อก่อนตัดสินใจจอง
    - ตรวจสอบข้อมูลการจองทั้งหมดก่อนส่งคำขอ

## Feature สำคัญ
- ระบบตรวจสอบวันว่างของชุด (จำลอง)
- ฟอร์มกรอกข้อมูลของลูกค้า(ชื่อผู้ใช้ในอินสตาแกรม และ ที่อยู่)
- หน้าข้อตกลงและเงื่อนไขการเช่า
- หน้าสรุปรายละเอียดการจอง

## Tech Stack
- React
- JavaScript
- CSS
- Tailwind
> โปรเจกต์นี้มุ่งเน้นการพัฒนา Frontend ยังไม่มีการเชื่อมต่อ backend

## Alignment with Krungthai Internship Objectives
โปรเจกต์นี้พัฒนาขึ้นโดยสอดคล้องกับเป้าหมายหลักของโครงการฝึกงาน Krungthai Bank ดังนี้:

- **การแก้ปัญหาจากสถานการณ์จริง**  
  โปรเจกต์เริ่มจากการวิเคราะห์ขั้นตอนการทำงานจริงของร้านเช่าชุดบน Instagram และออกแบบแนวทางแก้ไขที่สามารถนำไปประยุกต์ใช้ได้จริง

- **การพัฒนาทักษะด้าน Frontend อย่างเป็นระบบ**  
  แสดงให้เห็นถึงความเข้าใจในการออกแบบ user flow, การจัดการ state และการสร้าง UI ที่ใช้งานง่าย

- **แนวคิดแบบ User-Centered Design**  
  ออกแบบระบบโดยคำนึงถึงทั้งมุมมองของลูกค้าและร้านค้า เพื่อลดขั้นตอนที่ไม่จำเป็นและลดภาระงานซ้ำซ้อน

- **การคิดเชิงนวัตกรรมและการปรับปรุงกระบวนการทำงาน**  
  เปลี่ยนกระบวนการที่ต้องพึ่งพาการตอบข้อความแบบ manual ให้เป็นโครงสร้างดิจิทัลที่ชัดเจนและมีประสิทธิภาพมากขึ้น

- **ความพร้อมในการทำงานจริงในสภาพแวดล้อมองค์กร**  
  โปรเจกต์นี้สะท้อนแนวคิดการทำงานอย่างเป็นระบบ การสื่อสารที่ชัดเจน และการพัฒนาโซลูชันที่คำนึงถึงผลกระทบทางธุรกิจ
